user session

State that exists between a user logging on to AS ABAP and logging off again. During the user session, the user can work with AS ABAP. A user session is always associated with at least one ABAP session and a maximum of 16 ABAP sessions. The different types of user sessions are known as dialog sessions, background sessions, update sessions, RFC sessions, ICF sessions, and APC sessions. They are opened in different ways and are used for different purposes.
