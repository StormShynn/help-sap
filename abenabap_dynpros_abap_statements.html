ABAP Statements for Dynpros


The following statements can be used to call and process dynpros in an ABAP program:

  • CALL SCREEN
  • SET PF-STATUS
  • GET PF-STATUS
  • SET TITLEBAR
  • SUPPRESS DIALOG
  • LOOP AT SCREEN
  • MODIFY SCREEN
  • SET CURSOR
  • GET CURSOR
  • CONTROLS
  • REFRESH CONTROL
  • EXIT FROM STEP-LOOP
  • SET HOLD DATA
  • SET SCREEN
  • LEAVE [TO] SCREEN
