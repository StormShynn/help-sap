selection text

Part of the text elements of an ABAP program. A selection text or a text from ABAP Dictionary can be associated with every parameter or selection criterion defined using PARAMETERS or SELECT-OPTIONS. This text is then displayed alongside the corresponding input fields on the selection screen.
