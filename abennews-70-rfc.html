Remote Function Call for Release 7.0


1. New transactional RFC



Modification 1

New transactional RFC

A new interface has been developed for tRFC and qRFC, which will exist in parallel with the old interface as of Release 7.0. The statement for calling a transactional RFC via the new interface is CALL FUNCTION IN BACKROUND UNIT.

Note

Complete shipment from Release 7.0, EhP2.
