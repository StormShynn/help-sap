control

Software component of the ABAP runtime environment used to handle complex screen elements. Table controls, tabstrip controls, and splitter controls are platform-independent controls processed directly using dynpro statements and ABAP statements. In addition, there are platform-independent GUI controls accessed in ABAP by the classes in Control Framework.
