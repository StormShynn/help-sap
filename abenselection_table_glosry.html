selection table

Global internal table with a header line, the columns SIGN, OPTION, LOW, and HIGH and the name of a selection criterion. Used as an internal repository for selection conditions. Declaration with the statement SELECT-OPTIONS; can be evaluated in a comparison expression using the relational operator IN, where each row of the table represents a separate comparison.
