branch

Also known as selection. Control structure that can consist of multiple statement blocks defined using the keywords IF - ELSEIF - ELSE - ENDIF, CASE - WHEN - ENDCASE, TRY - CATCH - CLEANUP - ENDTRY and whose execution is dependent on conditions. Branches can be implemented in operand positions using conditional expressions.
