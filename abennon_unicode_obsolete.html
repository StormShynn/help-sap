Obsolete Disabling of the Unicode Check

When a program is created, the ABAP language version standard ABAP (Unicode) with enabled Unicode checks is configured by default.

  • Programs with the language version standard ABAP (Unicode) work in both Unicode systems and non-Unicode systems.
  • Programs with the language version obsolete ABAP (non-Unicode) only work in non-Unicode systems.

In the current SAP NetWeaver release, only Unicode systems are supported. Programs with the language version obsolete ABAP (non-Unicode) can no longer be executed in the current release. The ABAP language version cannot be reset to obsolete ABAP (non-Unicode). Higher ABAP language versions are possible because they cover the Unicode checks.
