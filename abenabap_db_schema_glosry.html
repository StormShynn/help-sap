ABAP database schema

Database schema assigned to the database user name used in an AS ABAP database read. A work process acting as a database user accesses the objects of this database schema implicitly. The database objects and implementations of AMDP methods managed by ABAP Dictionary are located in the ABAP database schema. The name of the ABAP database schema in AS ABAP can be determined using the function module DB_DBSCHEMA_CURRENT.
