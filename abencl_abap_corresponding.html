CL_ABAP_CORRESPONDING - System Class

The system class CL_ABAP_CORRESPONDING enables assignments of components between structures or between internal tables with dynamically specified mapping rules. Methods are available for simple assignments and for using a lookup table.

  • CL_ABAP_CORRESPONDING - Simple Assignment
  • CL_ABAP_CORRESPONDING - Lookup Table
