activatable checkpoint

Checkpoint that can be assigned to a checkpoint group using the addition ID of the statements ASSERT, BREAK-POINT, or LOG-POINT. The behavior of an activatable checkpoint is controlled either by group-specific or program-specific activation settings. Program-specific settings apply to all activatable checkpoints of a program and overwrite the settings of the current checkpoint group.
