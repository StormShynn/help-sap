function module exit

Customer exit for ABAP programs in the form of empty or partially implemented function modules delivered with the standard SAP system. Incorporated in ABAP programs using the statement CALL CUSTOMER-FUNCTION. The function module interfaces are predefined by SAP. The implementation can be empty or partially predefined. The customer must fully implement and activate them using transaction CMOD. It is advisable to replace function module exits by BAdIs.
