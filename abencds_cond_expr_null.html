ABAP CDS - cond_expr, NULL

Syntax

.... lhs IS [NOT] NULL ...

Effect

Identifies the null value in a CDS view. The relational expression is true if the value of lhs is (not) the null value. The same context-dependent rules apply to the operand lhs as to comparisons.
