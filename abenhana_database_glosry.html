SAP HANA Database

The SAP HANA database is the in-memory database of SAP HANA. It can be used as the database for an AS ABAP. In this case, HANA views can be accessed in ABAP programs with Open SQL - provided that these views are defined as external views in the ABAP Dictionary. Furthermore, AMDP can be used to manage and call database procedures written in SQLScript.
