CDS role

A CDS object defined using the statement DEFINE ROLE in the ABAP CDS CDS DCL. This object applies a CDS access control based on access conditions when CDS entities are accessed using Open SQL or SADL queries. Currently, every CDS role is a mapping role assigned to every user.
