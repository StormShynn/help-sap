transaction code

Twenty-character name associated with a dynpro, with another transaction code, or with a method of an ABAP program and which is used to execute programs. Transaction codes associated with dynpros are possible for executable programs, module pools, and function groups. Parameter transactions and variant transactions are associated with other transaction codes. Transaction codes associated with methods are possible for all program types that contain methods. Transaction codes are edited and managed using transaction code SE93.
