COMMIT WORK - Quick reference

Reference

Syntax

COMMIT WORK [AND WAIT].

Effect

Ends an SAP LUW; executes subroutines registered using PERFORM ON COMMIT and update function modules registered using CALL FUNCTION IN UPDATE TASK.

Addition

  • AND WAIT
    The current program waits for all high-priority update function modules to be executed.
