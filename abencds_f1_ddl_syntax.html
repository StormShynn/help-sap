ABAP CDS - DDL for Data Definitions

CDS data definitions in ABAP CDS are made using the data definition language CDS DDL in CDS source code. The CDS DDL follows the general CDS syntax rules and comprises the following statements for the definition and enhancement of CDS entities:

  • Statements for CDS views
  • DEFINE VIEW
  • EXTEND VIEW
  • Statements for CDS table functions
  • DEFINE TABLE FUNCTION
  • Language elements for various CDS entities
  • Input parameters
  • Typings
  • Element annotations

Notes

  • The CDS DDL for data definitions has its own editor in ADT, which is documented there. This can also be used to display the actual SQL DDL statement generated from the CDS DDL source text.
  • A further set of CDS DDL statements makes it possible to define metadata extensions.
