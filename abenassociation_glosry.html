association

Relationships between the nodes of a hierarchy. As a property of mesh nodes, a part of a mesh types for meshes. Associates mesh nodes as mesh paths. Furthermore, associations can be defined in the CDS DDL in ABAP CDS and evaluated there using path expressions.
