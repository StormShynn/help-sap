ABAP CDS - Annotations

An annotation enriches a definition in the ABAP CDS with metadata. It can be specified for specific scopes of a CDS object, namely specific places in a piece of CDS source code. SAP uses a set of predefined SAP annotations.

  • Syntax for Annotations
  • Scopes for Annotations
  • Evaluation of Annotations
  • SAP Annotations

Notes

  • Customers and partners are currently not advised to defined their own annotations and use only SAP annotations instead.
  • The annotations defined in the source text of a CDS entity can be supplemented or overridden with the annotations of CDS metadata extensions.
