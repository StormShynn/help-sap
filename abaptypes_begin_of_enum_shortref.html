TYPES BEGIN OF ENUM - Quick reference

Reference

Syntax

TYPES BEGIN OF ENUM enum_type [...].
  val1 [...], val2 [...], ...
TYPES END OF ENUM enum_type [...].

Effect

Defines an enumerated type enum_type.
