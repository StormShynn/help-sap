spool request

Output request sent to the SAP spool system. In a spool request, the pages of a spool list are passed to the SAP spool system. Spool parameters are defined when a spool request is opened and these parameters cannot be changed afterwards. A spool request is always assigned to the current spool list level. The spool requests of stacked spool list levels are also stacked.
