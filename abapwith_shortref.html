WITH - Quick reference

Reference

Syntax

WITH
  +cte1[( name1, name2, ... )] AS subquery[,
  +cte2[( name1, name2, ... )] AS subquery
  ... ]
  SELECT ... .
  ...
ENDWITH.

Effect

Open SQL statement. Introduces the definition of one or more common table expressions +cte1, +cte2 for use in a closing main query SELECT .... The results set of a common table expression is determined by a subquery subquery.
