pass by value

Method of passing data from actual parameters to formal parameters when the procedure is called, which is defined in the parameter interface of a procedure. In pass by value, a local data object is created as a copy of the actual parameter. Output parameters and return values are initialized when the procedure is called. Input parameters and input/output parameters are passed the value of the actual parameter when they are called. Changed formal parameters are only passed to the actual parameter if the procedure was concluded without errors - that is, whenever the last statement is reached or there is an exit using RETURN (or EXIT or CHECK). Refer also to pass by reference.
