INSERT field-group - Quick reference

Reference

Syntax

INSERT dobj1 dobj2 ... INTO { header | field_group }.

Effect

Obsolete: Uses global data objects dobj1, dobj2, ... to build the structure of a field group defined for the extract dataset of the program by FIELD-GROUPS.
