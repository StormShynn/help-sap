text environment

The text environment is part of the runtime environment of an ABAP program and is consists of a language, a locale, and a system code page. All programs in an internal session have a common text environment. By default, the text environment of an internal session is determined by the logon language and can be set by the statement SET LOCALE. The language of the current text environment is specified in the system field sy-langu.
