Unicode system

Single code page system in which characters are encoded in Unicode character representation. The system code page of a Unicode system is UTF-16 with platform-dependent byte order. The ABAP programming language supports the subset covered by UCS-2. In a Unicode system, all ABAP programs must have the ABAP language version Standard ABAP (Unicode). In the current SAP NetWeaver release, only Unicode systems are supported.
