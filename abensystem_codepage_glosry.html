system code page

Code page released for use on an application server. Each system in the actual release is part of a unicode-system, i.e. a single codepage system with the system codepage UTF-16 which has platform-specific byte sequence. Within this, the programming language ABAP supports the subset UCS-2 without surrogate. The system codepage of a single codepage system always also corresponds to the codepage of the text environment of an internal session.
