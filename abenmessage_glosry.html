message

Text that can be accessed using the statement MESSAGE. Messages are always edited in the logon language, which cannot be changed during a user session. Message texts are organized into message classes and message numbers and are edited in the transaction message maintenance. When a message is accessed using the statement MESSAGE, it is classified with a message type that defines the behavior of the message.
