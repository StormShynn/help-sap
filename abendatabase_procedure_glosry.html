database procedure

General synonym for stored procedure. In SAP HANA databases, name for an SQL Script procedure programmed in SQL Script or an L procedure programmed in L. Database procedures can be managed as AMDP procedures in the ABAP Managed Database Procedures (AMDP). See also database function.
