dialog step

State of a user session between a user action on the user interface of a dynpro and the sending of a new screen. During a dialog step, AS ABAP does not react to user actions. The current user session is assigned a work process of the application server which executes the program logic of the dialog step. Generally, the dialog step consists of the logic programmed for the PAI time of the current dynpro and PBO time of the following dynpro.
