SELECTION-SCREEN - SKIP

Quick Reference

Syntax

SELECTION-SCREEN SKIP [n] [ldb_additions].

Effect

This statement creates n blank lines on the current selection screen among the lines already filled. The value n must be specified directly as a single-digit positive number. If n is not specified, a single blank line is created.

The additions ldb_additions can only be used in the selection include of a logical database.
