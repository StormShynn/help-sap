Object Navigator

Development environment for the central editing of repository objects. Object Navigator displays repository objects as nodes of different tree representations (browser), such as Repository Browser, Repository Information System, or Enhancement Information System, in the navigation area. From here, they can be edited using forward navigation in the tool area. Called using transaction code SE80.
