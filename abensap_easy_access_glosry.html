SAP Easy Access

Initial program that is called after a dialog logon to AS ABAP. SAP Easy Access displays an initial menu for the access of a dialog user to AS ABAP. By default, this menu is either the SAP menu or a user menu. The SAP menu is an area menu which is centrally defined or overridden by an entry in the user master record. A user menu contains all the functions required by a dialog user and is created and assigned by the system administrator using the role maintenance (transaction PFCG). Each user can also define their favorites menu in SAP Easy Access that is displayed together with or instead of the default menu.
