sXML Library - XML Reader

The interfaces and classes of sXML library for theXML reader are:


IF_SXML   IF_SXML_NSURI_HELPER
  |         |
  |---------|
  |
  |--IF_SXML_READER
       |
       |--CL_SXML_READER
            |
            |--CL_SXML_STRING_READER
            |
            |--CL_SXML_TABLE_READER
            |
            |--CL_SXML_DATASET_READER
            |
            |--CL_SXML_HTTP_READER
            |
            |--CL_SXML_XOP_READER

The interface IF_SXML_READER contains the components that are required for every XML reader. The abstract class CL_SXML_READER implements all the general methods of an XML reader. The subclasses of these classes are specializations for different sources of XML data and provide a factory method for generating the associated XML reader.

The higher-level interfaces contain components for XML reader and XML writer:

  • IF_SXML contains constants for the supported XML format.
  • IF_SXML_NSURI_HELPER contains methods for namespaces.
